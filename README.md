# pairgame 
概要
　Entity（竹ブロック）で神経衰弱を行い、制限時間（20秒）内により多くのペアを見つけスコアを競うゲームです！

★初めに
　このリポジトリは、Java学習者である「あいうえお（Xアカウント：@aiueojava ）」が作成した
Minecraftプラグイン『pairgame（神経衰弱）』に関するものです。
デイトラ Javaコースで学んだ内容を活かして開発しました。

※ ご利用に関するトラブル等につきましては、一切の責任を負いかねますことを予めご了承ください。
※ Xの仕様により、未ログインの状態ですと投稿が表示されない場合があります。

★使い方
　①ゲーム画面にてコマンド「/gamestart」と入力することで開始されます。
 　なお、「/gamestart list」と入力することで過去の結果を参照することができます
  ②ブロックをクリックすると数字が表示され、一致するブロックを探します。
  　あたると点数が10点加算され、ブロックが消えます。
  特にペナルティはないのでどんどん押しましょう
